> **文档级别**: 交互设计
> **最后更新**: 2026-07-04

---

## 1. 新用户注册流程

```
首页
  ↓ 点击右上角「登录」
登录弹窗
  ↓ 点击「注册」切换
注册表单
  ↓ 填写: 用户名 + 邮箱 + 密码 + 确认密码
  ↓ 点击「发送验证码」
邮箱收到验证码
  ↓ 输入验证码
  ↓ 勾选「同意用户协议」
  ↓ 点击「注册」
注册成功
  ↓ 自动登录
首页 (已登录状态)
```

### 校验规则

| 字段 | 规则 |
|------|------|
| 用户名 | 6-25位，小写字母开头，仅小写/数字/下划线 |
| 邮箱 | 合法邮箱格式 |
| 密码 | ≥ 8 位，建议字母+数字 |
| 确认密码 | 与密码一致 |
| 验证码 | 6 位数字 |
| 用户协议 | 必须勾选 |

## 2. 登录流程

```
任意页面
  ↓ 点击右上角「登录」
登录弹窗 (居中)
  ↓ 输入: 用户名/邮箱 + 密码
  ↓ (可选) 勾选「记住我」
  ↓ 点击「登录」
  ├── 成功 → 返回来源页面 (redirectAfterLogin)
  └── 失败 → 显示错误，保留输入
```

### 手机号验证码登录

```
登录弹窗
  ↓ 切换到「手机号登录」
输入手机号
  ↓ 点击「发送验证码」
输入验证码
  ↓ 点击「登录」
```

### 忘记密码

```
登录弹窗
  ↓ 点击「忘记密码」
重置密码页
  ↓ 输入邮箱
  ↓ 发送重置链接到邮箱
邮箱点击链接
  ↓ 输入新密码
  ↓ 确认新密码
  ↓ 重置成功，跳转登录
```

## 3. 角色创建流程

```
导航栏 → 角色 → 创建
  ↓
步骤 1: 选择种族
  ├── 显示所有玩家种族卡片
  ├── 点击卡片查看详情 (属性修正、特质)
  └── 选中后点击「下一步」
  ↓
步骤 2: 选择专修
  ├── 显示 11 大类专修
  ├── 每类展开显示技能列表
  └── 选中后点击「下一步」
  ↓
步骤 3: 属性分配
  ├── 显示 4 项属性: 力量/敏捷/智力/精神
  ├── 基础值: 1d4 随机 (每项点击骰子)
  ├── 分配点: 10 点手动分配
  ├── 自动应用种族修正
  └── 确认后点击「下一步」
  ↓
步骤 4: 背景故事
  ├── 输入角色名称
  ├── 输入背景故事 (可选)
  └── 点击「完成创建」
  ↓
保存到云端
  ↓
跳转角色详情页
```

## 4. 角色编辑流程

```
个人中心 → 我的角色卡
  ↓
角色卡列表
  ↓ 点击「编辑」
角色编辑页 (同创建流程，预填数据)
  ↓ 修改字段
  ↓ 点击「保存」
  ├── 成功 → 显示「保存成功」
  └── 失败 → 显示错误信息
```

## 5. 规则浏览流程

```
导航栏 → 规则
  ↓
规则首页 (显示章节树)
  ├── 左侧: 章节导航 (可折叠)
  └── 右侧: 当前章节内容
  ↓ 点击章节
切换到对应章节
  ↓ 阅读规则
  ├── 悬浮术语 → 显示 tooltip
  ├── 点击关联规则 → 跳转
  └── 点击「下一章」→ 顺序浏览
```

## 6. 术语查询流程

```
导航栏 → 词条
  ↓
术语表页面
  ├── 搜索框: 输入关键词
  ├── 分类筛选: 核心/种族/神术/全部
  └── 术语列表 (字母排序)
  ↓ 点击术语
展开术语详情
  ├── 定义
  ├── 别名
  └── 关联词条 (可跳转)
```

## 7. 个人资料编辑流程

```
右上角用户区 → 个人中心
  ↓
个人主页
  ├── 身份信息区: 头像 + 保存按钮
  ├── 基本资料区: 昵称 + 性别 + 签名 + 保存按钮
  └── 账号安全区: 修改密码 + 绑定邮箱 + 绑定手机
  ↓
修改资料
  ↓ 点击「保存资料」
  ├── 成功 → 显示「资料已保存」+ 更新导航栏
  └── 失败 → 显示错误 + 保留输入
```

### 头像上传

```
点击头像区域
  ↓
选择文件 (jpg/png, ≤2MB)
  ↓
自动 Canvas 压缩到 200×200
  ↓
预览显示
  ↓ 点击「保存头像」
  ↓
上传到服务器
  ↓ 更新导航栏头像
```

## 8. 页面跳转与登录守卫

```
未登录用户访问 /profile
  ↓
重定向到 /login?redirect=/profile
  ↓
登录成功
  ↓
读取 redirect 参数，跳转回 /profile
```

### 全站登录状态保持

| 机制 | 说明 |
|------|------|
| localStorage | 存储 access_token / refresh_token |
| Axios 拦截器 | 每次请求附加 Bearer Token |
| 401 拦截 | Token 过期自动刷新或跳转登录 |
| 路由守卫 | 需认证页面检查 token |
| 导航栏同步 | 所有页面从 localStorage 读取登录状态 |
