> **文档级别**: 产品规划

## 1. 产品概述

Embers World 是原创 TRPG「灰烬世界」的官方数字平台，为玩家和 GM 提供规则查询、角色管理和世界观浏览服务。

## 3. 功能优先级

### P0 (必须有)

- 规则浏览（分类导航 + 内容展示）
- 术语表（搜索 + 悬浮提示）
- 种族/专修/神术展示
- 用户注册/登录
- 角色卡创建/编辑/保存
- 个人资料编辑

### P1 (应该有)

- 全文搜索
- 角色卡导出 (JSON/PDF)
- 世界观关联展示
- 游戏记录
- 后台内容编辑

### P2 (可以有)

- 收藏功能
- 浏览历史
- 角色卡分享链接
- 版本 diff 对比
- 通知中心

### P3 (以后再说)

- 交互式地图
- Mod 支持
- 开放 API
- 多语言

## 4. 非功能需求

| 指标 | 目标 |
|------|------|
| 首页加载 | < 2s (3G 网络) |
| API 响应 | < 200ms (P95) |
| 搜索响应 | < 500ms |
| 可用性 | 99.5% |
| 并发用户 | 500+ |
| 数据备份 | 每日自动备份 |
| 浏览器兼容 | Chrome 90+, Firefox 88+, Safari 14+ |
| 移动端适配 | 768px 以下可用 |

## 5. 验收标准 (关键功能)

### 5.1 规则浏览

- [ ] 可按章节导航浏览规则
- [ ] 每条规则显示: 标题、简介、正文、表格、相关规则、更新时间
- [ ] 规则中的术语可悬浮查看定义
- [ ] 可跳转到关联规则

### 5.2 角色卡

- [ ] 可选择种族（从数据库加载）
- [ ] 可选择专修（从数据库加载）
- [ ] 属性分配: 4 属性 1d4 随机 + 10 点手动分配
- [ ] 自动计算属性修正（种族加成）
- [ ] 可保存到云端
- [ ] 可编辑已保存的角色卡
- [ ] 可导出为 JSON 文件

### 5.3 用户系统

- [ ] 手机号验证码登录
- [ ] 邮箱注册（含验证码）
- [ ] 登录状态全站保持
- [ ] 个人资料: 昵称、签名、性别、头像
- [ ] 头像上传 (jpg/png, ≤2MB, 自动压缩)
- [ ] 退出登录
