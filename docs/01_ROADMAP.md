# Embers World Roadmap

## Alpha: Completed

- 001 project skeleton
- 002 importer

Alpha established the repository foundation, Django backend, Vue frontend, SQLite local development flow, structured rule data sources, and the minimal rule importer.

## Beta: Current Stage

- 003 Vue rulebook
- 004 category query
- 005 full text search
- 006 rule detail

Beta focuses on turning imported rule data into a usable product experience through API-backed Vue pages.

## Gamma: Later

- lore
- divine arts
- arcane arts

Gamma expands the rulebook beyond the minimal core rule set and starts organizing larger world and magic systems around the same data source and API pattern.

## Delta: Later

- character sheets
- builder
- campaigns

Delta starts after the rule API and Vue rulebook are stable enough to support durable character and campaign systems.
